        Sudoku Rules

        Fill the 9x9 grid so that each column, each row, and each of the nine 3x3 subgrids contain all of the digits from 1 to 9.

        Multiplayer Scoring
        • +50 Points for each correct cell placed.
        • -20 Points & 1 Strike for a mistake.
        • 3 Strikes and you are eliminated!
        Useful Features
        • Keyboard Ready: Use arrows to navigate and numbers to input! Backspace to erase. U for Undo, H for Hint.
